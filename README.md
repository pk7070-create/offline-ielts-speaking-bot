# Offline IELTS Speaking Bot

A voice-based IELTS speaking practice system built using Python. This project simulates a real IELTS interview environment where users answer spoken questions and receive automated prompts.

## Features

- Voice-based interaction
- Computer asks IELTS speaking questions
- Speech recognition for user responses
- Text-to-speech for interviewer voice
- Continuous mock interview flow
- Fully offline (no API required)
- Modular design for future AI integration

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3

## Future Scope

- AI-based answer evaluation
- Band score prediction
- Grammar feedback
- Performance analytics
